

Проек CarShop (Продажба на автомобили и авточасти).
Проектът е базиран на Windows Forms Application.
Първата форма е Login Form -> форма за влизане в програмата, посредством четене на потребителски данни от файл.
Започнах и втори вариант на логване, който не съм довършил. Искам да изпозлвам interface Iuser.

Потребители за работа със системата записaни във файла:
petar 1234
user 2345
user1 2222


След логване се отваря -> MainForm, тази форма е препратка към допълнителните форми.
Двете форми ShopForm и Tires са съотвтно за продажба на автомобили и гуми.


Формата Clients сега започвам работа по нея. Там мисля да изведа списък с клиенти и да имам опция да добавям и изтривам.
В тази форма мисля да използвам алгоритъм за сортиране.

Формата Specification представлява информативна страница, с препрадки към Web страници с информация съответния автомобил.
Препрадката се осъществява с натискане върху изображението.
